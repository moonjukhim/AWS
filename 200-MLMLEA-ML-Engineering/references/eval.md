###

| 하이퍼파라미터      | 설명            | 값 변화에 따른 영향 |
|-------------------|----------------|--------------------|
| alpha             | L1 정규화 계수   | 값 ↑ → 가중치 희소성 증가, 과적합 방지 |
| eta               | 학습률 (learning rate) | 값 ↑ → 빠르게 학습, 과적합 위험 증가 |
| max_depth         | 트리 최대 깊이 | 값 ↑ → 모델 복잡도 증가 |
| min_child_weight | 리프 노드가 분할되기 위한 최소 가중치 합 | 값 ↑ → 보수적 분할, 과적합 감소 |
| num_round | Boosting 반복 횟수 (트리 개수) | 값 ↑ → 학습 반복 증가, 성능 향상 가능 (과적합 주의) |

---

##### 성능 급락 지점 존재

원인 
- 부적절한 하이퍼파라미터 조합 (예: 너무 깊은 트리, 너무 낮은 eta)
- Early stopping에 가까운 실패 케이스
- 과적합/과소적합 설정
