# XGBoost 

XGBoost를 사용하여 개인의 연소득이 5만 달러를 초과할 확률을 예측하는 방법을 보여줍니다. 표준 UCI 성인 소득 데이터셋을 사용합니다. 
XGBoost와 같은 그래디언트 부스팅 머신(Gradient Boosting Machine) 방법은 다양한 양상의 테이블 형식 입력 데이터를 사용하는 이러한 유형의 예측 문제에 있어 최첨단 기술입니다. Tree SHAP( arXiv 논문 )은 트리 앙상블 방법의 SHAP 값을 정확하게 계산할 수 있도록 하며, C++ XGBoost 코드베이스에 직접 통합되었습니다. 
다음의 데이터에서 연소득이 5만 달러를 초과하는지 예측하는 XGBoost 모델을 만들어줘.
