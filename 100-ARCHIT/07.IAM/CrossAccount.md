## Task2.사용자 생성

  - iamtest
    - 인라인 정책 지정

```json
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "VisualEditor1",
            "Effect": "Allow",
            "Action": "sts:AssumeRole",
            "Resource": "arn:aws:iam::[AWS account ID]:role/NetworkAdmin"
        }
    ]
}
```
