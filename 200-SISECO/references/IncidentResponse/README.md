| 방법                               | 장점                      | 단점                     | 적합한 경우             |
|-----------------------------------|---------------------------|--------------------------|--------------------------|
| **Lambda**                        | 간단하고 빠름             | 실행 시간 제한(15분)     | 경량 인시던트 처리       |

| **CodePipeline + CFN**            | IaC 기반 자동화           | 느릴 수 있음             | 새 환경 자동 생성 시     |
